LOL Esports Draft Dashboard
This is a draft dashboard for League of Legends Esports. The data comes courtesy of Oracle's Elixir's public Lol Esports dataset. To get started, select champions for each role on the blue and red team. As champion matchups are populated win rates in those matchups as well as winrates by year will be shown to the right of the draft dashboard. Select statistics to plot on the X and Y axes as well as champions to plot stats for. If year or patch is selected in for the x-axis, the draft dashboard will show the y-axis stat with respect to the year/patch for each champion in the selected matchup. If another stat is selected for the x-axis, the draft dashboard will show the y-axis stat with respect to the selected stat for champion 1 in the selected matchup with the size and color of the dots corresponding to the number of occurences and winrate respectively.
